# Changelog

## [0.1.0] - 2026-03-19
- Initialized MVP monorepo skeleton for BIOCARD Aviation TMS rebuild.
- Added FastAPI backend with domain entities and RBAC-ready auth stubs.
- Added React + TypeScript + Ant Design frontend shell with core screens.
- Added Docker Compose infrastructure (PostgreSQL, Redis, backend, frontend, Nginx).
- Added project documentation, local run instructions, and deployment baseline.
- Added live planning workbench API with demo seeding, filtering, split/merge, AWB assignment, flight assignment, and fixation checks.
- Added API-backed planning screen with filters, selection, details drawer, and dispatcher actions.
